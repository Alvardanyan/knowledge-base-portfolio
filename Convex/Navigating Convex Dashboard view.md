# Navigating Convex Dashboard view

## Overview

The Dashboard serves as your gateway to deeper insights and enhanced control over your
business operations. With organizational features and detailed activity logs, you can monitor
your and your team's performance and track every action.

## Who uses this feature

* Managers, regional managers, representatives, and salespeople
* Primarily for Commercial Service and Commercial Construction business types

## Feature configuration

* Contact [marketing@convexlabs.io](mailto:marketing@convexlabs.io) to set up a demo and get more information about Convex.

## Explore Dashboard views

The Dashboard is the starting page for all users. It is the first feature section on the navigation
panel. There are two views of the dashboard depending on your role:

- Dashboard Overview
- My Work
- Reports
- Activity Logs

> **Note:** Both tabs are available to user and manager roles, the starting page is different.

- Workflow Dashboard: Users and reps start the dashboard on this page.
- Analytics Dashboard: Managers start the dashboard on this page.

## Overview of Analytics Dashboard

The Analytics Dashboard in Atlas provides a comprehensive view of the team's activity across
three key areas: properties, contacts, and engagement with contacts. The data displayed
depends on the features enabled, such as Accounts and Opportunities.

The view comprises a bar chart and a table. A stacked bar chart visualizes all activities for selected
users over a specified period, while a detailed table presents the same metrics in a structured
format.

### Chart

You can change the view presented on the chart by:

- Date range: Click the date range dropdown and select the range you want to view. You can
  also create a custom range.

- Legend: Adjust the view of a chart by clicking and hiding the data set by color. You can
  select more than one data set to hide. Clicking again shows the data set.

- Export: Click Export to download the table data as displayed.

### Table

You can change the view of the table by:

- Filtering by users:

  1. Hover over and click the three dots next to the Name column.

  2. From the popup, select the user.

- Sorting the columns: Drag and drop the column to reorganize the table.

> **Note:** The chart and table are interconnected: adjusting one (e.g., date range, filtered users)
updates the other in real-time.

## Explore Dashboard subsections

There are four or five subsections available to you, depending on your role.

- Overview: Contains Workflow Dashboard and Analytics Dashboard views and it is the
  default screen.

- My Work: Contains filtered views for property, campaign, and opportunity assignments.

- Reports: Contains activity-based reporting around engagement, property, and
  opportunities.

- Activity Logs: Contains user and team activity history.

- Manage Users: Appears for users who have manager role permissions enabled.

### Overview subsection

The Overview subsection includes the following sections:

- Metric cards: Each metric card provides a trend comparison against the previous 30 days,
  showing whether activity has increased or decreased. This helps you track performance
  over time. A trend only appears once 30 days of data have been collected.

> **Tip:** Hover over each card to learn more about it.

- Team Property Actions: Any view, assignment, or update to a property. This includes
  any activity such as starring or archiving the property, account associations, opportunity
  associations, card creations/updates, target associations, campaign and rep
  assignments, stage updates, property notes, custom field updates, or CRM pushes.

- Team Contact Actions: Any view, assignment, or update to a contact. This includes any
  activity such as saving to a property, adding to a list, account associations, opportunity
  associations, notes on the contact, or target associations.

- Team Phone Calls: Any phone calls placed inside Atlas or from the mobile app.

- Team Emails: Any sending or viewing of emails on the web or through the mobile app.
  This includes sending emails outside of Atlas to a saved contact.

> **Note:** Emails and phone calls from the mobile app must be initiated from the Atlas
mobile app.

- Personal work: Shows all your starred and archived properties.

- My Team's Work: Shows all your team's properties with their prospect stages.

- My Campaigns: Shows all campaigns - private and shared.

- Property Assignment: Shows all users and the number of assigned properties to each of
  them.

- Opportunities by Stage: Shows the number of opportunities at each stage.

- Account Owners: Shows the names of account owners with their respective accounts and
  opportunity numbers.

### My Work subsection

The My Work subsection includes the following sections:

- Property Assignment: Shows all the reps with the number of assigned properties. When
  viewing this tab, your name is selected in the filter as a default user, and the data is
  reflected in the table.

- Campaigns: Shows all the campaigns that can be viewed.

- Prospecting Stage: Shows all the opportunity stages with the respective number of
  properties.

- Starred & Archived: Shows all the starred and archived properties.

### Reports subsection

Reports subsection includes the following sections:

- Engagement: Includes a chart and a table that shows the team's activities such as phone
  calls, properties viewed, contacts shared, etc for the selected period.

- Prospecting: Shows stage activity for properties by each user.

- Opportunity: Shows stage activity for opportunities by each user.

> **Note:** To use this feature Accounts and Opportunities must be enabled.

- Leaderboard: Shows a table based on property activity level.

### Activity Logs subsection

The Activity Logs subsection displays the history of all activities done by the sales rep including
date and data filters such as:

- Campaigned: Shows the campaigns created, assigned, and unassigned by reps.

- Notes: Shows the notes created and edited by reps.

- Assigned: Shows the reps assigned and unassigned to a property.

- Changed Stage: Shows the stage selected by reps.

- Contact added: Shows the contacts added by reps to a property.

- Custom Fields: Shows the actions that happened in custom fields by reps.

### Manage Users subsection

The Manage Users subsection includes the following sections:

- Users: Shows the name of the users in the organization.

- Email: Shows the email address of each user.

- Role: Shows the roles of each user.

- Days Active Last 30 Days: Shows the number of days in the last 30 days that the user took
  an action.

- Activity Ranking: Shows the total number of property, contact, and engagement actions the
  user took in the last 30 days.

- User Actions: Shows the rank placement of the user on the number of property, contact,
  and engagement actions in the last 30 days.

- CSV Exports: Shows the number of exports performed by the user.

- Contact Exports: Shows the number of exports by the user if it is enabled.
