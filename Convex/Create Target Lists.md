# Create Target Lists

## Overview

Create Target Lists to automatically identify prospect contacts, saving time and improving prospecting efficiency.

## Who uses this feature

* Managers, regional managers, representatives, and salespeople
* Primarily for Commercial Service and Commercial Construction business types

## Feature configuration

* Contact [marketing@convexlabs.io](mailto:marketing@convexlabs.io) to set up a demo and get more information about Convex.

## Create a List

1. Go to the Navigation panel > Map.

2. Use Map filters on the property list panel to find properties.

3. After you have the list that you are comfortable working with, click View Properties.

4. Click Select All to include all the properties in a list. Otherwise, select properties individually.

5. From the Actions menu that appears, select Add to target list.

6. From the Add to Target List pop-up that opens, you can:

   a. Add the targets to the existing target list. Select the target list and click Add to List.

   b. Create a new target list:

      i. Click +New List.

      ii. Enter a target list name and click Create List and Add.

7. After you create a new target list, the Target List created message appears at the top. Click View to open it.

8. On the New Target List Created pop-up that opens, select Yes if you want to populate targets with contacts.

9. On the Populate Leads pop-up that opens, select the criteria for the leads.

10. When finished, click Proceed. Populate Leads pop-up
