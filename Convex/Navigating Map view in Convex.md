# Navigating Map view in Convex

## Overview

Use the Atlas Map view to visualize and explore sales opportunities. It highlights each
commercial property within your territory, providing an estimated total of buildings within each
population center.

## Who uses this feature

* Managers, regional managers, representatives, and salespeople
* Primarily for Commercial Service and Commercial Construction business types

## Use Map navigation tools

The navigation tools help you efficiently explore and move around your selected territories.

1. In the navigation panel, click Map. The Map screen opens.

2. On the side of the map, you can find these tools:

Convex provides three map views to tailor your navigation experience:

| Icon | Name | Description |
| --- | --- | --- |
| & | Map View | The default mode that displays the map using clean graphics for simplified navigation and analysis. |
| © | Satellite View | Overlays real satellite imagery onto the map, offering a detailed, realistic visual of the area. |
| K] | Combo View (Auto) | Automatically switches between Map and Satellite views based on your zoom level, delivering the most relevant visual for your current perspective. |

> **Note:** When you hover over the view icons, a tooltip informs you of the mode you will
activate by clicking. For example, in the default Map View, hovering over the Satellite icon
will display the tooltip "Lock the satellite view." Clicking it switches to Satellite View.

| Icon | Name | Description |
| --- | --- | --- |
|  | Add a new property | Use this to add a property to Atlas so that you can track your activity on a property not available by default. |

1. Click Add new property.

2. On the Add a new property screen that opens, click PROCEED. A new pin appears.

3. Place the pin at the desired location to add a custom property.

4. In the pop-up that opens, edit the address of the property as needed.

5. When finished, click Create.

| Icon | Name | Description |
| --- | --- | --- |
| 🗺️ | Map layers | Use this to see the selected campaigns represented on the map. |

1. Click Map layers.

2. On the Map layers pop-up that opens, select the campaigns you want to display.

3. When finished, click Apply.

For example, you want to see all the campaigns for Louisville Industrial. After you select it,
you will see red dots on the map, displaying the campaigns with their corresponding color. By
zooming in, you can see property boundaries in the campaign color and click on the dots to
access detailed information about each property. This feature helps you easily identify and
manage properties within your sales territories.

| Name | Description |
| --- | --- |
| Zoom in | Use this for a closer and more detailed view. |
| Zoom out | Use this to adjust the view for a larger area or broader perspective. |
| Fullscreen view | Expands the map to an entire screen view. |
| Pegman | Drag the icon onto the map to open the street view. To exit, click the back arrow on the top left corner of the screen or press Esc on your keyboard. |
| Tilt map | Use this to adjust the viewing angle of the map for a three-dimensional perspective. By tilting the map, you can gain insights that are not apparent from a standard rooftop view. |

For more, see Adjust the view with Tilt Map.
