# View properties list panel in Convex

## Overview

The properties list panel displays a list of properties within your selected area. It allows you to
view, filter, and access specific property information, helping you quickly locate and manage
properties within your sales territories.

## Who uses this feature

* Managers, regional managers, representatives, and salespeople
* Primarily for Commercial Service and Commercial Construction business types

## Feature configuration

* Contact [marketing@convexlabs.io](mailto:marketing@convexlabs.io) to set up a demo and get more information about Convex.

## View properties list panel in Convex

In Atlas, the properties list panel is located in the Map section. It includes the Search bar, Filters,
property results, and property list cards.

For more, see Use Map Search and View Map Filters.

Under Properties, you can find the total number of buildings in the selected region.

> **Note:** When you zoom in or out, the number of properties adjusts dynamically to reflect the
properties within the visible territory.

| Icon | Name | Description |
| --- | --- | --- |
|  | Lock list | Locks the list to the current list of properties. This helps maintain focus on a specific set of properties without updating the list dynamically based on map adjustments. |
|  | Sort list | Sorts the list by different characteristics. It allows you to organize properties from largest to smallest based on: <br><br>• Recommended <br>• Stories <br>• Building (ft²) <br>• Grounds (acres) <br>• Year Built |

## Property list card

The property list card displays the overview of each building. Hover over the card to get a
preview of the property attributes:

- Type (Office)

- Stories (38)

- Area (234 ft²)

- Year Built (1982)

Click the card to open the property details panel (PDP). You can find more information about the
property in the Property Attributes section in PDP.

## Manage properties in a properties list panel

When you are satisfied with the list, you can select properties to manage.

1. Hover over the building for the circle to appear and click to select it.

> **Tip:** To include all the buildings in a properties List, click SELECT ALL. You can select up
to 2000 properties at a time.

2. To remove the selection, click CLEAR.

3. After you select the properties in the list, you are ready to use them. Click the Actions
dropdown and select one of the following options:

- Assign to your board

- Assign to campaign

- Update stage

- Assign to rep

- Add to target list

  a. Assign to your board

This action assigns a task (card) for this property to your board.
It includes selecting the status for that card.

i. Select the status of the property.

ii. When finished, click APPLY.

  b. Assign to campaign

Campaigns are used to create groups (or lists) of properties. They are an organizational tool that
can be managed in a workflow. Users can filter various tables in Convex by campaign assignment.
Users can also choose to highlight campaigns on the map view.

  c. Update stage

This helps represent where the prospect stands, allowing you to follow up and measure progress
more effectively.

i. Click the opportunity stage for the building.

ii. When finished, click APPLY.

  d. Assign to rep

This action links the sales representative to the property, allowing them to track all activities
related to that property.

i. Select the rep you want to assign the property.

ii. When finished, click APPLY.

  e. Add to target list

This allows you to view and update the status of the opportunity, track key actions, and manage
any related tasks or interactions.

i. On the Select a Target List pop-up, choose a list to add or click + New List to
create a new one.

ii. When finished, click Add to List.

> **Note:** The feature is in beta and not available to everyone.
