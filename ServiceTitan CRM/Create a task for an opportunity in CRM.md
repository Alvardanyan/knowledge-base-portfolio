# Create a task for an opportunity in CRM

## Overview

Assign tasks to the opportunities in every stage of your sales pipeline to keep them organized and track your to-dos.

## Who uses this feature

•  Administrators, managers, sales managers, and customer service representatives (CSRs)

•  Primarily benefits Commercial Service and Replacement business types

•  Applies to all trades

## Feature configuration

•  This feature is currently in Early Access and available for specific accounts. It is subject to change. If you want to enable this feature for your account, reach out to your Customer Success Manager (CSM).

## Create a task

1. Go to the navigation bar and click CRM > Opportunities.
2. Select the opportunity you want to create a task for.
3. On the Opportunities drawer that opens, click Add Task.
4. Enter the task details and click confirm.

If there are no open tasks in the Next Task column, create a task by using an inline entry.

1. Hover over and click anywhere on the Next Task column.
2. Enter the task description and click confirm.

## Add information to the task

After creating the task, add extra details to keep it clear and organized.

1. On the Opportunities drawer, hover over the open task and click All fields.

2. On the drawer that opens, you can:

• Edit a task description.

• Select a Type from the list.

• Set a Due Date.

• Select an Assignee from the list.

• Enable the toggle for Mark as High if the task is a priority.

3. When finished, click Save.

> **Tip:** You can also change the task type by clicking the task type icon next to the task description on the opportunity drawer.

## Complete a task

You can complete a task from the Opportunities table or the Tasks section. For more, see Manage tasks for opportunities in CRM.

### Complete a task from the Opportunities table

1. Go to the navigation bar and click CRM > Opportunities.

2. From the Opportunities table > Next Task column, click the square to complete a task.

After the task is completed, it is removed from the Next Task column.

> **Note:** You can also complete the task from the Opportunities drawer > Open Tasks section.

## Edit a task

You can edit a task from the Opportunities screen or the Tasks section. For more, see Manage tasks for opportunities in CRM.

### Edit a task from the Opportunities table

1. Go to the navigation bar and click CRM > Opportunities.

2. On the Opportunities table > Next Task column, hover over anywhere and click to edit it.

3. After you finish, click confirm to save the changes.

### Edit a task from the Opportunities drawer

1. Go to the navigation bar and click CRM > Opportunities.

2. Click the opportunity you want to edit a task.

3. From the Opportunities drawer that opens, click the task to edit it.

4. When finished, click confirm to save the changes.
