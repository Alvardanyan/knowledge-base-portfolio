# Service Agreements Auto-renewing This Month report

## Overview

The Service Agreements Auto-renewing This Month report provides a view of the service agreements that are set to automatically renew in the current month. This report helps you monitor and manage upcoming auto-renewals,
including details like agreement ID, customer name, total price, and estimated gross margin, and helps you efficiently track and oversee service agreements.

## Who uses this feature

- Administrators, project managers
- Primarily benefits Commercial Service and Replacement and Commercial Construction business types
- Applies to all trades

## Feature configuration

- Account configuration is required to use this feature. Please contactTechnical Support for details.

## Things to know

- This report is built to show agreements auto-renewing during the current month, however you can reset the filters to change the report results if
needed.
- If you have access to custom reporting, you can edit the report columns to display additional information. Click Edit Report to add or remove columns. If you're unsure which columns you want to include in your
report, use the Reporting Dictionary to search for appropriate data fields.

## Benefits of using this report

- Proactive renewal management: You can identify all service agreements set to auto-renew within the current month. This allows you to proactively communicate with customers, address any potential changes or
cancellations, and ensure a smooth renewal process, preventing unexpected service interruptions or revenue loss.
- Revenue forecasting and planning: By viewing the total agreement price and estimated gross margin of auto-renewing agreements, you can gain
insights into upcoming revenue. This data assists in financial forecasting, resource allocation, and strategic planning.
- Customer relationship management: The report provides customer names and location details for auto-renewing agreements. This lets you tailor communication and service offerings based on specific customer needs, improving customer satisfaction and retention.

## Run report

The Service Agreements Auto-renewing This Month report is a built-in report
that's available to run without any required setup.

## Locate the report

1. Go to the navigation bar and click Reports.
2. In the Search field, enter Service Agreements Auto-renewing This Month,
then click the report card.

## Filter the report

This report has the following filters automatically set:

- Status: The report is configured to display only agreements in the Activated and Auto-renew statuses.
- Filter By: The report filters based on the Agreement Start/End Date.
- From - To: The report displays the current month.
- Deferred Revenue Export Status: The report filters for all export statuses of invoices with deferred revenue.

You can set additional filters to help narrow down your report results:

- Use the Business Unit field to select the job business units (BUs) you want to report on. You can select one or multiple BUs. By default, this filters for all BUs.
- Use the Account Manager field to select the Account Manager assigned to the agreement.
- Use the Deferred Revenue As of Date to set the date you want the deferred revenue for invoices calculated on before.
When you're finished setting filters, click Run Report.

## Read report results

Your report results display in a table based on the filters you set. By default, the Service Agreements Auto-renewing This Month report includes the following columns:

- ID: Unique ID for the service agreement
- Name: Name of the service agreement
- Customer Name: Name of the customer on the service agreement
- Location Names: Name of the location(s) on the service agreement
- Total Agreement Price: Total price of the service agreement
- Gross Margin $ (Estimated): The estimated profitability of the service agreement in dollars
- Gross Margin % (Estimated): The estimated profitability of the service agreement as a percentage
- Start Date: Start date of the service agreement
- End Date: End date of the service agreement
- Duration Length (Months): Duration of the service agreement in months
- Auto-renew: Whether auto-renew is set to Enabled or Disabled for the service agreement
- Billing Schedule: The Billing Schedule set for the agreement
- Created On: The date the agreement was created
- Created By: The name of the employee who created the agreement
- Created By Method: How the service agreement was created

> **Tip:** Click Edit Columns to add and remove columns. You can arrange report columns and apply filters for further customization.
